主要功能列表 LIST OF KEY FEATURES

多图文菜单

用户输入自定义关键字，系统将自动回复已设置的多图文菜单。

关注时推送图文

用户关注公众号后第一时间推送图文菜单，全面展示企业产品和服务信息，拉近企业与用户距离。

会员卡

在微信内植入电子会员卡功能，帮助企业建立集品牌推广、会员管理、营销活动、统计报表于一体的微信会员管理平台。

高级会员注册

根据会员全面资料，归类客户会员类型，提供定制化服务。

一键关注

快捷简便的关注方式，迅速吸纳企业粉丝群体。

个性微官网

通过简单直观的设置，快速自定义不同行业的专属微官网，系统提供海量模版供用户选择搭配，并配有不同图文设计风格和高级自定义功能，打造独特个性风格。

照片墙

为用户提供图片存储和展示服务。通过个性相册的创建，发布精选照片发布，传播活动咨询，提升互动效应。

微信支付

新移动支付模式，帮助企业形成营销闭环。

视频文件

通过多媒体视频展示内容，丰富用户体验。

APP菜单

提供微信公众号自定义菜单管理功能，用户无需再通过输入关键词触发回复，直接点击菜单就可以看相关的内容。同时提供嫁接企业原有APP桥梁，提供个性化功能拓展，增强用户粘性。
